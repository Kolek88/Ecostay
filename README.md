# EcoStay Web Application

**Live Demo:** [View the Live Production Site](http://cvkhaliq.atwebpages.com/ttpEcostay/landing.html)

## Disclaimer
The project is still ongoing so there may be many bugs in the website or incomplete functions/pages

## Project Overview
EcoStay is a web-based platform designed to promote sustainable practices and manage environmental initiatives. Built with a full-stack PHP architecture, the application features an interactive dashboard, user authentication, and an administrative backend for tracking reports and gamifying sustainable actions through a reward system.

## Features
* **User Authentication:** Secure login and registration tailored for standard users and administrators.
* **Interactive Dashboards:** Dynamic PHP scripts (`dashboard_data.php`, `get_analytics.php`) that fetch and display real-time metrics.
* **Gamification System:** Built-in logic to grant points and claim rewards (`grant_points.php`, `claim_reward.php`) based on user actions.
* **Database Integration:** Structured SQL database connectivity for persistent data storage and secure administrative controls.

## Environment Configuration
**Important Note:** The source code in this repository is configured for a local development environment. 
* Database credentials in `db_connect.php` have been sanitized for security. 
* To run locally, configure a local server (e.g., XAMPP) and update the credentials to match your local SQL database. 
* The Live Demo utilizes a separate, secure production database.

## Technologies Used

* **Backend:** PHP
